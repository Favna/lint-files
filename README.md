# klasa-community-plugins-lint

Central shareable lint files. This is inteded for private use, but you can use it via:

```bash
npm i --save-dev KlasaCommunityPlugins/lint-files
```

## eslint rules

```json
{
    "extends": "@kcp/lint"
}
```

## tslint rules

```json
{
    "extends": "@kcp/eslint-config-lint"
}
```
